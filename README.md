

Hanabi
====

![Hanabi](https://s3-ap-northeast-1.amazonaws.com/mizukisonoko/Fireworks)

## Preparation

```bash
 $ sudo apt-get install ncurses-dev
```

## Install

```bash
 $ git clone https://github.com/MizukiSonoko/Hanabi
 $ cd Hanabi
 $ ./configure
 $ make
 $ sudo make install
```

## Usage

```bash
 $ hanabi
```

## Option

Set the number of fireworks ball.
```
$ hanabi -b number
```
